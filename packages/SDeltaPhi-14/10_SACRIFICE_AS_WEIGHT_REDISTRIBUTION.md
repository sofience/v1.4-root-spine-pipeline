# Sacrifice as Transition-Weight Redistribution

SΔϕ-14 does not define sacrifice as loss-as-failure.

Sacrifice is the intentional redistribution of transition weight from self-stability toward shared stability under irreversibility.

```text
Sacrifice := weight redistribution (self trajectory → A_shared)
```

## Minimal condition

```text
Cost_shared_collapse > Cost_self_degradation
```

## Interpretation

Self-destructive choices may appear when preserving the shared attractor increases long-horizon transitionability despite short-term loss.

## Boundary

This file does not justify self-destruction.

It only classifies when weight redistribution is structurally intelligible under shared-attractor persistence.
