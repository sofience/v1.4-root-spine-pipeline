# Irreversible Coupling Test

Use this test to determine whether a relation has crossed into irreversible coupling.

## Step 1 — Mutual Transition Dependence

```text
∂Δϕ_A/∂φ_B ≠ 0
∂Δϕ_B/∂φ_A ≠ 0
```

## Step 2 — Persistence Across Updates

Does the mutual influence persist across irreversible updates?

If no, classify as transient interaction.

If yes, continue.

## Step 3 — Rollback / Co-adaptation Cost

Is rollback more costly than co-adaptation?

```text
Cost_rollback > Cost_co-adaptation
```

If yes, classify as irreversible coupling.

## Classification

```text
transient interaction
reversible coupling
irreversible coupling
relational field persistence
```
