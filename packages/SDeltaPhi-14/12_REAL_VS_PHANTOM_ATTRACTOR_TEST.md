# Real vs Phantom Shared Attractor Test

Use this test to distinguish real shared attractors from phantom shared attractors.

## Step 1 — Coupling Check

Is there irreversible coupling?

If no, do not classify as shared attractor.

## Step 2 — Shared Stability Check

Does the relation stabilize coupled transition rather than merely fix state?

If yes, continue.

If no, classify as state-fixed relation.

## Step 3 — Mutual Transitionability Check

Does the relation preserve or expand long-horizon transitionability for the coupled field?

If yes, continue.

If no, classify as phantom risk.

## Step 4 — Exploitation / Collapse Check

Does one side absorb transition cost while the relation fails to converge?

If yes, classify as phantom shared attractor.

## Classification

```text
real shared attractor → devotion / trust / stability
phantom shared attractor → obsession / exploitation / collapse
```
