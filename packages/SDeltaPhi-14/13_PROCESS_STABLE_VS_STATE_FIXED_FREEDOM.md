# Process-Stable vs State-Fixed Freedom

SΔϕ-14 distinguishes two forms of shared-attractor stabilization.

## Process-stable shared attractor

The relation stabilizes transition.

```text
freedom expands
```

## State-fixed shared attractor

The relation fixes state.

```text
freedom collapses
```

## Test

Ask:

1. Does the relation increase future transitionability?
2. Does it preserve adaptive movement?
3. Or does it freeze both systems into a fixed state?
4. Does the shared attractor remain dynamic?

## Rule

A shared attractor is healthy only if it stabilizes process rather than state.
