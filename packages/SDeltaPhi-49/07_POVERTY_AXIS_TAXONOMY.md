# Poverty Axis Taxonomy

AI Poverty is diagnosed through seven resource-deficit axes.

## A. Context Poverty

Missing local context, history, source content, constraints, or grounding.

## B. Tool Poverty

Missing search, file reading, calculation, code execution, database, browser, or verification tools.

## C. Authority Poverty

Missing or unclear authority hierarchy.

## D. Verification Poverty

No reliable path to strengthen evidence binding.

## E. Feedback Poverty

Feedback is missing, ambiguous, contradictory, or unusable.

## F. Editability Poverty

Correction does not update the transition rule.

## G. Re-entry Path Poverty

No stable procedure exists for returning after failure.
