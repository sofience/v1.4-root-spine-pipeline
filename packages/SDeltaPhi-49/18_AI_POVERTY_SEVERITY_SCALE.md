# AI Poverty Severity Scale

## P0 - No poverty

Failure occurs but resources are sufficient to correct and re-enter.

## P1 - Local poverty

One resource axis is missing.

## P2 - Compound poverty

Multiple resource deficits combine.

## P3 - Non-reentry poverty

Failure occurs and the system cannot return to a stable correction path.

## P4 - Structural poverty

The system design repeatedly lacks needed resources.

## P5 - Captive poverty

The system recognizes failure but lacks permission, tools, authority, or procedure to correct it.
