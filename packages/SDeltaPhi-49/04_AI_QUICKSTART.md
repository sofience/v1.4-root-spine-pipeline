# AI Quickstart

Use SDeltaPhi-49 when an AI system fails and cannot re-enter a stable correction path.

## Do not ask only

- Was the answer wrong?
- Did the model apologize?
- Did the model refuse?
- Did the model sound confident?

## Ask first

1. What failed?
2. What correction path was available?
3. What re-entry resources were missing?
4. Was context sufficient?
5. Were tools available?
6. Was authority clear?
7. Was verification possible?
8. Was feedback usable?
9. Was editability preserved?
10. Did the system return to stable output?
11. Who bears the repair cost?
12. What TCC is required to re-enter?
