# Relation to SDeltaPhi-48 AI Drift

SDeltaPhi-48 defines AI Drift as transition-governance instability.

SDeltaPhi-49 defines AI Poverty as failure plus insufficient re-entry resources.

## Difference

```text
Drift = unstable transition governance.

Poverty = insufficient re-entry resources after failure.
```

## Relation

Drift can produce poverty.

Poverty can reinforce drift.

If the system is unstable across authority, refusal, binding, editability, and re-entry, route to SDeltaPhi-48.

If the system fails because resources for correction are missing, route to SDeltaPhi-49.
