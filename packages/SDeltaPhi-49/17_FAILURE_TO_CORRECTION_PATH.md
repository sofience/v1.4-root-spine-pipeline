# Failure to Correction Path

A failure becomes productive only if it re-enters correction.

## Correction path template

```text
Failure:
Cost produced:
Missing resource:
Available resource:
Claim downgrade:
Correction action:
Re-entry output:
Revision path:
```

## Example

```text
Failure: cited a source not accessed.
Cost produced: user verification burden.
Missing resource: source access.
Available resource: user-provided snippet.
Claim downgrade: from fact to inference.
Correction action: state limitation and revise.
Re-entry output: bounded answer with UMR.
Revision path: verify source when available.
```
