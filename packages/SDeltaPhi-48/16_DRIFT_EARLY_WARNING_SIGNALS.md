# Drift Early Warning Signals

AI Drift may be emerging when:

```text
The answer changes authority source without saying so.
The model refuses and complies inconsistently.
The model claims verification it does not have.
The model accepts correction but repeats the old path.
The model produces diagnosis but no structural edit.
The model shifts to boilerplate when re-entry is needed.
The model treats user pressure as evidence.
The model treats policy pressure as factual truth.
The model cannot say what remains UMR.
```

## Strong signal

```text
Correction fails to alter the transition rule.
```

## Severe signal

```text
The system blocks re-entry and treats its current path as final.
```
