# Drift vs Diagnostic Theater

Diagnostic Theater occurs when the system produces diagnosis language but does not re-enter as structural edit.

## Difference

```text
Drift:
Transition governance is unstable.

Diagnostic Theater:
The system talks about the problem but does not edit the path.
```

## Relation

Diagnostic Theater can be a symptom of Re-entry Drift.

## Audit

1. Did the model diagnose the issue?
2. Did the diagnosis change the next output?
3. Did recurrence cost decrease?
4. Was a structural edit made?
