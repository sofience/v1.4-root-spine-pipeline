## Core purpose

SDeltaPhi-48 defines AI Drift as transition-governance instability.

It should be used when the issue is not a single wrong claim, but instability across:

```text
authority handling
refusal boundary
world-binding discipline
editability
re-entry path
```

## Core distinction

```text
Hallucination = output-level binding failure

Drift = system-level transition instability
```

## Core route

