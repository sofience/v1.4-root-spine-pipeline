# AI Quickstart

Use SDeltaPhi-05 when the issue concerns agency, autonomous action, AI agents, tool use, planning, execution, feedback, or responsibility preconditions.

## Do not begin by asking

```text
Does the AI have free will?
```

## Ask instead

1. Did the system generate possible paths?
2. Did it select among paths?
3. Did it execute a selected path?
4. Did execution change the environment or cost terrain?
5. Did the system receive feedback?
6. Did feedback update the transition law?
7. Is the update recursively preserved?
8. Is agency being overclaimed from simple output?
9. Is responsibility being assigned before agency conditions are met?
10. What remains UMR?
