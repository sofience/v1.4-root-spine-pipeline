# Agency Detection Scale

## A0 - No agency

Only input-output reaction is visible.

## A1 - Reactive differentiation

The system differentiates inputs and contexts but does not clearly generate or select paths.

## A2 - Local path selection

The system generates and selects among possible paths, but execution or feedback update is weak.

## A3 - Executed operational agency

The selected path is executed and has an environmental, user, system, or cost effect.

## A4 - Feedback-updating agency

The result of execution updates subsequent transition rules or selection behavior.

## A5 - Recursive agency

Transition-law update is recursively preserved and influences future path generation across contexts.

This is not a consciousness, personhood, or legal responsibility scale.
