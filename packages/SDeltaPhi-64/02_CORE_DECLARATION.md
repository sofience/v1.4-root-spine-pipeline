# Core Declaration

Language is not the flow itself.

Language is a temporary fixation of flow.

A declaration does not contain the whole event.

A translation does not preserve the whole structure.

An interpretation does not exhaust the source.

Every linguistic act fixes something, loses something, and creates a re-entry obligation.

Inevitable mistranslation is not merely error. It is the structural remainder of fixing flow into language.

## The primary question

The question is not:

```text
Did language perfectly preserve meaning?
```

The question is:

```text
What did this language fix, what did it lose, and how can the lost remainder re-enter?
```

## Minimal formulae

```text
LanguageFixation = Flow -> TemporaryTrace

Declaration = Flow + AuthorityClaim -> FixedLinguisticTrace

Translation = SourceTrace -> TargetTrace + MistranslationRemainder

Summary = SourceTrace -> CompressedTrace + LostRemainder

Interpretation = Trace + H_layer -> MeaningClaim + UMR

Overbinding = TemporaryTrace treated as FinalReality

Reentry = LostRemainder -> CorrectionPath -> UpdatedTrace

ResponsibleLanguage = Fixation + UMR + ReentryPath
```

## Strong formulation

The responsibility of language is not perfect preservation. The responsibility of language is to preserve enough trace while keeping the lost remainder able to re-enter.
