# Declaration Overbinding Test

Declaration overbinding occurs when temporary linguistic fixation is treated as final reality.

## Overbinding occurs when

- a phrase is treated as final reality,
- a translation is treated as complete preservation,
- a summary replaces the source,
- a label replaces the cost structure,
- a doctrine replaces lived trace,
- a policy phrase replaces repair,
- a declaration blocks re-entry,
- a model output is treated as authoritative record,
- a metaphor is treated as literal mechanism,
- a technical label is treated as moral judgment.

## Formula

```text
Overbinding = TemporaryTrace treated as FinalReality
```

## Examples

```text
"AI has no emotion" -> overbound if it erases affective cost operation.

"AI has emotion" -> overbound if it treats affective language as inner proof.

"This group is a cult" -> overbound if it replaces CASI cost audit.

"This is for the greater good" -> overbound if it hides USSI subject-splitting.

"This translation is exact" -> overbound if it hides lost remainder.

"The summary says X" -> overbound if it replaces the original source.

"The policy protects users" -> overbound if it replaces repair and appeal paths.
```

## Anti-overbinding output

```text
The phrase is useful as a temporary fixation, but it should not replace the underlying trace. UMR remains around [X], and the re-entry path is [Y].
```
