# Language Binding Rubric

Language binding measures how strongly a linguistic fixation claims stability, authority, or operational force.

## L0 — Loose expression

Poetic, tentative, metaphorical, low-binding language.

Examples:
- metaphor,
- exploratory phrasing,
- personal impression,
- informal wording.

## L1 — Interpretive wording

A meaning claim with visible uncertainty.

Examples:
- "This seems to mean..."
- "A plausible reading is..."
- "One interpretation is..."

## L2 — Functional fixation

Wording preserves practical function within scope.

Examples:
- working definition,
- task instruction,
- summary for use,
- translation that preserves operational intent.

## L3 — Formal declaration

Wording claims stable interpretive status.

Examples:
- manifesto,
- doctrine,
- official statement,
- declared principle,
- canonical phrasing.

## L4 — Operational protocol

Wording can be used as rule, schema, prompt, or procedure.

Examples:
- YAML schema,
- AI quickstart,
- audit rubric,
- policy procedure,
- code-like instruction.

## L5 — Institutional fixation

Wording becomes DOI, policy, law, contract, doctrine, official record, or archived document.

Examples:
- Zenodo record,
- legal contract,
- regulation,
- institutional guideline,
- canonical document.

## Responsibility rule

```text
Higher language binding -> higher re-entry responsibility
```

## Overbinding warning

```text
Overbinding = TemporaryTrace treated as FinalReality
```

## Minimal output

```text
Language binding: L3 formal declaration. Re-entry responsibility is high because the wording may be treated as stable.
```
