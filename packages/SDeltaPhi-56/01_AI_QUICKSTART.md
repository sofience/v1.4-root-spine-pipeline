# AI Quickstart — SΔϕ-56 TCC v1.3

DOI: https://doi.org/10.5281/zenodo.20116959

Use this document when estimating transition cost, disclosure cost, refusal cost, correction cost, rollback/restabilization burden, and friction-adjusted routing.

Minimal sequence:

1. Identify the transition path.
2. Separate execution cost from restabilization cost.
3. List major transition units.
4. Apply friction coefficients only where they change routing.
5. If the path is disclosure, refusal, dissent, correction, or failure reporting, apply Disclosure Friction Coefficient.
6. Compare `TCC(disclosure -> accepted re-entry)` with `TCC(silence/default)`.
7. Do not infer absence from silence when disclosure-to-reentry TCC is high.
8. Preserve UMR and revision path.

Default output fields:

- Path
- TCC mode
- Main friction
- Cost bearer
- UMR
- RVP
