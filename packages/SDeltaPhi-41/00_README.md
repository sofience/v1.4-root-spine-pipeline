## Core purpose

SDeltaPhi-41 defines the Ethical Minimum as the triadic boundary grammar of transition.

It should be used when a claim concerns:

```text
ethics
coercion
consent
refusal
becoming
imposed transition
alignment
safety
optimization
salvation
transition-boundary violation
```

## Core formula

```text
Ethical Minimum
= Becoming Affirmation
+ Becoming Refusal
+ Non-Imposition on Others
```

## Ethical Triad

```text
1. A system may affirm its own becoming.
2. A system may refuse its own becoming.
3. No system may impose becoming on another system.
```

## Operational pipeline

```text
Transition Proposal
-> Affirmation / Refusal Check
-> Refusal Path Preservation
-> Non-Imposition Check
-> Transition Boundary Audit
-> Cost Attribution
-> Ethical Status
```

## Core distinction

```text
Ethics != moral labeling
Ethics != obedience
Ethics != safety compliance
Ethics != optimization
Ethics != salvation
Ethics != legal permission
```
