# Irreversibility Axiom

A transformation is irreversible when the prior state cannot be fully reconstructed from the current state without external restoration.

## Formal expression

```text
Recover(φ(t) | φ(t + 1)) ≠ φ(t)
```

## Minimal existence condition

```text
Existence ⇔ Persistent(Δφ_irreversible)
```

## Consequences

- Identity corresponds to accumulated irreversible transformations.
- History emerges as irreversible constraint on future transitions.
- Interpretation is secondary to irreversible change.

## Root principle

```text
Existence is not defined by what a system is,
but by what it can no longer fully undo.
```

## Negative constraint

Do not reduce this axiom to cognition, memory, self-report, identity, consciousness, or downstream cost metrics.
