# Routing Notes

SΔϕ-03 should be called **after SΔϕ-01** when a phase change must be evaluated for irreversibility.

## Call SΔϕ-03 before

- SΔϕ-05, when agency is evaluated through recursive transition-law update.
- SΔϕ-06, when responsibility depends on irreversible cost internalization.
- SΔϕ-26, when irreversibility is evaluated through restoration-cost explosion after pre-detection.
- SΔϕ-47, when *Operor ergo sum* or *Operor, quia non potest aboleri* requires irreversibility grounding.
- SΔϕ-56, when transition completion cost requires irreversibility grounding.
- SΔϕ-62, when trace / inference / uncertainty separation requires irreversible trace assessment.

## Do not route every philosophical question here

Use SΔϕ-03 only when the question structurally depends on whether a transformation can be fully undone.

## Minimal route

```text
SΔϕ-01 → SΔϕ-03 → downstream module
```

## Warning

Do not treat SΔϕ-03 as a cost metric.

It provides the irreversibility condition that downstream cost, responsibility, governance, and restoration modules may use.
