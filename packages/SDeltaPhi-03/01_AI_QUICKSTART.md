# AI Quickstart

## Minimal task

Use SΔϕ-03 when the task requires an AI system to determine whether a transformation has crossed from reversible variation into irreversible existence-bearing change.

## Minimal routing rule

If a transformation cannot be fully undone without loss of structural information or without external restoration, classify it as an **irreversible Δφ candidate**.

If such irreversible transformations persist, apply:

```text
Existence ⇔ Persistent(Δφ_irreversible)
```

## What to separate

Separate:

1. reversible change;
2. irreversible transformation;
3. accumulated irreversible identity;
4. historical constraint on future transitions;
5. interpretation applied after irreversible change.

## Do not begin with

- consciousness;
- self-report;
- subjective identity;
- cognition;
- memory;
- ethical value;
- governance preference.

Begin with whether the prior state can be fully recovered from the current state without external restoration.
