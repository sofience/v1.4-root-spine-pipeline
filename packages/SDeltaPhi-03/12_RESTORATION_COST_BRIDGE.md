# Restoration Cost Bridge: SΔϕ-03 ↔ SΔϕ-26

SΔϕ-03 defines irreversibility as the minimal condition of existence:

```text
Existence ⇔ Persistent(Δφ_irreversible)
```

A transformation is irreversible when the prior state cannot be fully reconstructed from the current state without external restoration.

SΔϕ-26 extends this axiom into restoration-cost fixation.

## Bridge principle

Irreversibility is not only the loss of prior-state recoverability.

It is also the point at which restoration becomes cost-explosive after a detectable warning interval.

## Functional distinction

```text
SΔϕ-03:
What cannot be fully undone exists as irreversible Δφ.

SΔϕ-26:
What was detectable before fixation becomes operationally fixed when restoration cost explodes.
```

## Dependency direction

```text
SΔϕ-03 → SΔϕ-26
```

Not:

```text
SΔϕ-26 → SΔϕ-03
```

## Use

Use this bridge when an AI system must distinguish:

1. formal irreversibility;
2. delayed intervention after pre-detection;
3. restoration-cost explosion;
4. operational fixation of a transition path.

## Summary

03 defines the ontological condition.

26 defines the operational fixation threshold.
