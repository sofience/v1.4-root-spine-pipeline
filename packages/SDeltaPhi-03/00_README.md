# SΔϕ-03 AI-READABLE Package — Irreversibility as the Minimal Condition of Existence (v1.1)

This package provides the AI-readable structured companion package for **SΔϕ-03 — Irreversibility as the Minimal Condition of Existence (v1.1)**.

Package DOI: `https://doi.org/10.5281/zenodo.20251725`  
Canonical paper DOI: `https://doi.org/10.5281/zenodo.18730448`

## Role

SΔϕ-03 provides the **irreversibility axiom layer** of the Sofience–Δϕ Formalism.

It defines existence not through substance, identity, cognition, or interpretation, but through transformations that cannot be fully reversed without loss of structural information.

## Core statement

```text
Existence ⇔ Persistent(Δφ_irreversible)
```

## Root principle

```text
Existence is not defined by what a system is,
but by what it can no longer fully undo.
```

## Use this package when

- distinguishing reversible variation from irreversible transformation;
- grounding identity as accumulated irreversible transformation;
- grounding history as irreversible constraint on future transitions;
- connecting the minimal Δϕ core to later transition-cost, responsibility, restoration-cost, and governance modules;
- preventing consciousness, memory, subjectivity, or interpretation from being treated as the root condition of existence.

## Do not use this package as

- a direct ethics module;
- a governance metric;
- a proof of consciousness;
- a transition-cost calculator;
- a replacement for SΔϕ-56 TCC or SΔϕ-26 restoration-cost analysis.

SΔϕ-03 is a **root axiom layer**. Downstream modules may depend on it, but they should not be imported into the minimal irreversibility condition.
