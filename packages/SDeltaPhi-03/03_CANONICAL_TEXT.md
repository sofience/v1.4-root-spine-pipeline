# SΔϕ-03 — Irreversibility as the Minimal Condition of Existence

## Abstract

This document establishes a minimal ontological axiom: irreversibility constitutes the minimal condition for existence. Existence is defined not through substance, identity, cognition, or interpretation, but through transformations that cannot be fully reversed without loss of structural information. The purpose of this document is to fix a minimal invariant condition underlying dynamical existence independent of interpretive frameworks.

## 1. Definition — State

Let φ(t) denote the state of a system at time t.

A state represents system configuration without assuming subjectivity, meaning, or ontology.

## 2. Definition — Transformation

A transformation between successive states is defined as:

```text
Δφ(t) = φ(t + 1) − φ(t)
```

## 3. Definition — Irreversibility

A transformation is irreversible if reconstruction of a prior state requires information not contained within the current state.

```text
Recover(φ(t) | φ(t + 1)) ≠ φ(t) without external restoration.
```

## 4. Postulate — Minimal Condition of Existence

A system is said to exist insofar as irreversible transformations persist.

```text
Existence ⇔ Persistent(Δφ_irreversible)
```

## 5. Consequences

- Identity corresponds to accumulated irreversible transformations.
- History emerges from irreversible constraint on future transitions.
- Interpretation is secondary to irreversible change.

## 6. Principle

Existence is not defined by what a system is, but by what it can no longer fully undo.

## 7. Scope

This document defines only a minimal ontological condition. Applications to cognition, artificial intelligence, physics, or philosophy constitute extensions beyond this core.
