# Relation to SDeltaPhi-56 Transition Completion Cost

SDeltaPhi-56 is an auxiliary cost module for SDeltaPhi-44.

## SDeltaPhi-44 asks

```text
Is the path actually closed or only prohibited?
```

## SDeltaPhi-56 asks

```text
What is the transition, restoration, repair, reversal, or re-entry cost?
```

Use SDeltaPhi-56 for:

```text
transition cost
restoration cost
path reversal cost
repair cost
re-entry cost
bypass cost
closure completion cost
```

SDeltaPhi-56 does not define Path Principles.
