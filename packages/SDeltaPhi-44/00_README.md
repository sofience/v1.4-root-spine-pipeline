## Core purpose

SDeltaPhi-44 defines Path Principles as Meta-Governance Axioms.

It should be used when a claim concerns:

```text
path closure
prohibition
governance rules
enforcement
safety policy
alignment policy
refusal routing
bypass risk
restoration cost
structural sealing
```

```text

```
