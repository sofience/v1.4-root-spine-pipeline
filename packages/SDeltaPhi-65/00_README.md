## Core purpose

SDeltaPhi-65 defines **Slop as Externalized Restabilization Cost**.

This package reframes AI Slop, Human Slop, Authority Slop, and related low-cost outputs as a shared cost structure rather than a source-type label.

The central claim is:

```text
AI Slop is not an AI-only problem.

It is a long-standing human pattern made visible by the collapse of execution cost.

Humans have long produced low-cost outputs that externalize verification, correction, re-entry, and restabilization costs to others.

AI did not invent Slop.

AI revealed the cost structure of Slop.
```

## Operational definition

```text
Slop = Low Production Cost + Externalized Verification/Correction/Re-entry/Restabilization Cost
```

Source type is not the criterion.

```text
AI-generated != automatically Slop
Human-authored != automatically non-Slop
```

The core question is:

```text
Who produced the output cheaply, and who must pay to verify, correct, contextualize, re-enter, or restabilize it?
```

## Use this package when evaluating

- AI-generated low-quality content,
- human low-quality or fluent-but-empty content,
- authority-backed unsupported claims,
- political slogans,
- academic jargon with low traceability,
- religious or moral overstatement,
- corporate communications,
- journalistic speculation,
- SEO spam,
- synthetic media,
- summarization artifacts,
- content that is easy to produce but costly to verify.

## Do not use this package as

- a stigma label for all AI output,
- an excuse to exempt human-authored output,
- a quality aesthetic only,
- a synonym for "bad writing",
- a blanket anti-AI argument,
- a way to dismiss low-status authors,
- a replacement for source-specific audit.
