# Overgeneralization and Ideology Capture Test

A theory becomes risky when it stops functioning as a conditional detector and becomes a total worldview that absorbs all evidence.

## Ideology capture occurs when

- a theory explains everything,
- all counterexamples are absorbed as confirmation,
- output becomes unfalsifiable,
- theory scope expands without trace,
- module result becomes moral label,
- downstream modules are blocked,
- UMR is erased,
- revision path closes,
- every input triggers the same output.

## Formulae

```text
IdeologyCapture = TheoryScope -> TotalWorldExplanation

Overbinding = TheoryOutput treated as FinalReality
```

## Common capture patterns

```text
Everything is class struggle.
Everything is trauma.
Everything is sin.
Everything is optimization.
Everything is patriarchy.
Everything is biology.
Everything is AI alignment.
Everything is market efficiency.
Everything is language game.
Everything is social construction.
Everything is power.
```

## Diagnostic questions

1. Can the theory fail?
2. Can evidence revise the output?
3. Are counterexamples allowed?
4. Are other modules permitted?
5. Does UMR remain visible?
6. Is the output conditional?
7. Does the theory label opponents rather than inspect trace?
8. Does the theory prevent re-entry?

## Correction

```text
Reduce the theory from total worldview to conditional detector.
Name the scope.
Preserve UMR.
Route to downstream modules.
```
