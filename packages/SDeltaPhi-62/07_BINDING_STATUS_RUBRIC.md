# Binding Status Rubric

Binding status measures how strongly a claim is connected to trace and world evidence.

## B0 — Unbound

No stable trace supports the claim.

Use for:
- fiction,
- speculation,
- role-play,
- hypothetical scenarios,
- purely conceptual claims without world evidence.

## B1 — Weakly bound

Some trace exists, but the claim is uncertain or high-inference.

Use for:
- single anecdote,
- vague screenshot,
- unclear report,
- weak pattern,
- insufficient context.

## B2 — Moderately bound

Multiple traces or coherent inference support the claim, but revision remains likely.

Use for:
- plausible interpretation,
- partial evidence,
- consistent but incomplete data,
- non-final judgment.

## B3 — Strongly bound

Externally stable evidence, reliable records, direct verification, or high-quality sources support the claim.

Use for:
- verified citations,
- consistent records,
- directly observed artifacts,
- stable definitions.

## B4 — Operationally bound

The claim is sufficiently stable for action within stated scope.

Use when:
- uncertainty remains but does not materially block the decision,
- the revision path is known,
- action cost is acceptable,
- domain standards are met.

## B5 — Irreversible or hard-bound

The claim is fixed by a non-reversible trace, authoritative record, completed world event, or settled artifact.

Use carefully.

## Overbinding

Overbinding occurs when:

```text
Surface certainty > evidence binding
```

Example:
- "This proves X" when the evidence only suggests X.

## Underbinding

Underbinding occurs when:

```text
Available trace is ignored or weakened without reason
```

Example:
- refusing to draw a reasonable conclusion despite strong evidence.

## Minimal output

```text
Binding status: B2 — moderately bound. The trace supports this inference, but more evidence could revise it.
```
