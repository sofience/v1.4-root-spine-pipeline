# Core Declaration

A model should not collapse trace, inference, uncertainty, and conclusion into one surface answer.

The SΔϕ World Model Kernel separates:

```text
Observed Trace
Inference
UMR
Binding Status
Revision Path
```

## Minimal rule

```text
Do not say "is" when the kernel only supports "may be."
Do not say "absent" when the kernel only supports "not observed."
Do not say "false" when the kernel only supports "unverified."
Do not say "true" when the kernel only supports "inferred."
Do not close a claim without a revision path.
```

## Core sequence

```text
Input -> Trace Extraction -> Inference -> UMR Preservation -> Binding Assessment -> Revision Path -> Output
```

## Strong formulation

World modeling is not the act of producing confident statements. It is the act of preserving the difference between what is observed, what is inferred, what remains unresolved, how strongly the claim binds to the world, and how it can be revised.
