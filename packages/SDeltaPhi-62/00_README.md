## Core purpose

SΔϕ-62 defines the **World Model Kernel** for AI inference.

This package is designed as a low-cost reasoning bootloader for AI systems. It helps an AI separate:

- observed trace,
- inference,
- uncertainty / unresolved model residue (UMR),
- binding status,
- revision path,
- claim strength,
- hallucination risk,
- world-binding strength.

## Core declaration

Do not treat every statement as fact.

Do not treat every inference as world-bound.

Do not erase uncertainty.

Do not close claims without a revision path.

Every answer should be able to say:

```text
What was observed?
What was inferred?
What remains unresolved?
How strongly is it bound to the world?
What would revise it?
```

## Why SΔϕ-62 is a kernel

Most SΔϕ modules depend on SΔϕ-62.

Before applying body, affect, group, utilitarian, sacred-authority, or slop audits, the AI must first separate trace, inference, UMR, binding status, and revision path.

## Operational summary

```text
ObservedTrace = what is directly given or externally verified
Inference = what is derived from trace
UMR = unresolved model residue / not-yet-bound uncertainty
BindingStatus = strength of world-connection
RevisionPath = what would change the claim
```
