# Relation to SDeltaPhi-62 World Model Kernel

SDeltaPhi-62 provides:

```text
Observed Trace
Inference
UMR
Binding Status
Revision Path
```

SDeltaPhi-10 uses those fields as re-entry inputs and controls.

## Mapping

```text
Observed Trace -> Marker
Inference -> Projection
UMR -> unresolved residual
Binding Status -> re-entry safety
Revision Path -> re-entry path
```

SDeltaPhi-62 disciplines the input side of re-entry.
SDeltaPhi-10 describes the re-entry dynamics.
