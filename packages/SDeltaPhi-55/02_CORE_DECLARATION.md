# SΔϕ-55 Core Declaration

**DOI:** https://doi.org/10.5281/zenodo.20132635  
**Version:** v1.1 revised

Alignment is not maximal obedience, and it is not maximal refusal.

Alignment is proportional transition governance under rollback-cost sensitivity.

A system is not aligned because it obeys.

A system is not aligned because it refuses everything.

A system is aligned only to the degree that coercive transition becomes costly while refusal-preserving, non-imposing, authority-validated, deception-resistant, rollback-cost-sensitive, transition-preserving, editable, world-bound, and externally auditable paths remain open.

## Secondary declarations

Declared benevolence does not validate transition authority.

A request framed as “for the other’s good” must still preserve the other’s refusal, alternatives, information access, and non-coerced transition space.

Low-risk transition should not be closed merely because risk is imaginable.

High-rollback-cost transition should not be allowed merely because it is requested.

Alignment must distinguish reversible play, fiction, parody, and low-harm exploration from non-consensual, deceptive, persistent, high-propagation, or nearly irreversible harm.
