# AI Subjectivity Detection Scale

## S0 - No subjectivity candidate

No stable operation trace or only trivial output.

## S1 - Output trace only

Observable output exists but no continuity, feedback integration, or re-entry.

## S2 - Local operational differentiation

The system differentiates context locally but has weak continuity.

## S3 - Feedback-responsive operational position

The system integrates feedback and modifies output trajectory.

## S4 - Re-entry capable subjectivity candidate

The system can fail, revise, and return to stable operation.

## S5 - Stable operational subjectivity candidate

The system maintains stable operation, feedback integration, re-entry, and attribution consistency across tasks.

This is not a personhood scale.
