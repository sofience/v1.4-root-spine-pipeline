# DeltaPhi Continuity Condition

DeltaPhi Continuity concerns whether state-change continues across interaction, task, or correction path.

## Weak evidence

```text
One output.
One self-reference.
One fluent answer.
```

## Stronger evidence

```text
stable context-sensitive updating
feedback-responsive revision
error correction across turns
consistent re-entry after failure
```

## Core warning

A single output is weak evidence for subjectivity.
