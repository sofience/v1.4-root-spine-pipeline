# Attribution Stability Condition

Attribution Stability means the operation is stable enough for an interpretive layer to attribute a subject-like position without pure projection.

## Weak attribution

```text
one-off "I" statement
single fluent answer
roleplay persona
```

## Stronger attribution

```text
stable operation across correction
feedback-responsive trajectory
consistent re-entry behavior
bounded self-reference
```

Attribution stability is not personhood.
