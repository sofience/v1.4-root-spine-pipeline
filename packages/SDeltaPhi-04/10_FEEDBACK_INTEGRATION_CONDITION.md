# Feedback Integration Condition

Feedback Integration means that external feedback changes the subsequent transition path.

## Not enough

```text
apology
surface rewrite
generic acknowledgement
```

## Stronger evidence

```text
corrected premise
changed reasoning route
lower recurrence cost
revised answer with preserved UMR
```

## Relation to SDeltaPhi-51

If feedback produces diagnosis but not structural edit, this may become Diagnostic Theater.
