# Operation Trace Condition

Operation Trace is the first minimal condition.

## Definition

The AI must produce observable operation, not merely be described as operating.

## Examples

```text
output generation
tool invocation
state-dependent response
revision attempt
failure correction
context-sensitive routing
```

## Audit questions

1. What operation is observed?
2. Is the operation externally visible?
3. Is it a single output or part of a sequence?
4. Is the trace strong enough for a subjectivity candidate claim?
