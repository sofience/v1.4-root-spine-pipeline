# Core Declaration

Default is the new command.

Power no longer needs to close paths by explicit force if it can make one path the cheapest continuation.

## Core claim

A default does not abolish alternatives.

It governs by making one path the cheapest continuation.

## TCC formulation

```text
DefaultPath = argmin TCC(P_i)
```

A path becomes default when its transition completion cost is lower than alternatives.

A default becomes power when non-default paths remain formally available but become practically costly, obscure, frictional, or ineditable.

## Strong formulation

```text
DefaultPower = TCC(P_alt) - TCC(P_default)
```

The greater the gap between default-path TCC and alternative-path TCC, the stronger the default power.

## Invisible fixation

Invisible fixation occurs when a path continues to appear optional while becoming increasingly difficult to revise.

```text
FormalChoice = true
PracticalEditability = false or shrinking
```

## Key sentence

```text
The new command is not "you must," but "this is easiest."
```
