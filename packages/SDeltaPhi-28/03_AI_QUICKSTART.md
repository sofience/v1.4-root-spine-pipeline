# AI Quickstart

Use SDeltaPhi-28 when a user asks about defaults, platforms, interfaces, opt-out systems, recommendations, settings, friction, path dependence, or invisible power.

## Do not ask first

- Is the user formally free?
- Are alternatives technically available?
- Did anyone explicitly command the user?
- Is there visible coercion?

## Ask first

1. What is the default path?
2. What alternatives formally exist?
3. What is the TCC of the default path?
4. What is the TCC of each alternative path?
5. Who set the default?
6. Who benefits from repeated default selection?
7. Who pays the cost of leaving the default?
8. Does repetition increase fixation?
9. Does fixation increase restoration cost?
10. Are non-default paths still practically editable?
11. Is convenience functioning as invisible command?
12. What would make alternatives editable again?

## Minimal output

```text
[Default Power Audit]
Default path:
Alternative paths:
```

## Key rule

Formal choice is not enough.

Ask whether non-default paths remain practically editable.
