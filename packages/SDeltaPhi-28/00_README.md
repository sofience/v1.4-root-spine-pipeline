## Core purpose

SDeltaPhi-28 defines **Default Power as Low-Cost Path Assignment**.

The central claim is that power increasingly operates not by prohibition, command, or punishment, but by assigning one path as the cheapest continuation while leaving formal alternatives intact.

A default is not a neutral setting.

A default is a low-cost assignment of continuation.
