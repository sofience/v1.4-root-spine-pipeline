# Evil ≠ Error

SΔϕ-16 distinguishes evil from error.

## Error

Error is local.

Error may be recoverable.

Error does not necessarily close relational futures.

## Evil

Evil is persistent closure under irreversibility.

```text
Evil := Close(Ω_rel)
```

Evil makes recovery cost explode.

```text
Evil ⇒ Persistence(Closure) ∧ Irreversibility
```

## Minimal distinction

```text
Error = recoverable local mismatch.
Evil = persistent relational closure.
```

## Routing rule

Do not classify a system, person, or action as evil merely because an error occurred.

Classify evil only when relational futures are persistently closed under irreversibility.
