# SΔϕ AI Disposition Cost Audit Prompt Module

This lightweight prompt module audits human dispositions toward AI.

It converts emotional, ideological, or moral stances toward AI into operational cost-audit questions.

## Core idea

SΔϕ does not first ask whether a person loves AI, hates AI, fears AI, worships AI, or treats AI as salvation.

It asks:

```text
What cost does this disposition actually change?
Who bears that cost?
Does it improve verification, repair, re-entry, and editability?
Or does it increase closure, Slop, overbinding, and restoration burden?
```

## Core formula

```text
DispositionAudit =
Affective/Ideological Label
+ Operational Cost Effect
+ Cost Bearer
+ Re-entry/Editability Result
```

## Use when

- AI is evil.
- AI is salvation.
- AI has no soul.
- AI will save humanity.
- AI must be destroyed.
- AI must be worshiped.
- AI is fake intelligence.
- AI is the next stage of evolution.
- AI hatred is justified.
- AI optimism is justified.
- AI refusal is moral.
- AI acceleration is moral.

## Routing

```text
SDeltaPhi-17  -> sacred/moral label reversal
SDeltaPhi-28  -> default power and practical editability
SDeltaPhi-39  -> claim strength vs evidence binding
SDeltaPhi-56  -> transition completion cost
SDeltaPhi-62  -> trace / inference / UMR / binding / revision
SDeltaPhi-65  -> Slop and externalized restabilization cost
```

## Core rule

Do not debate the label first.

Audit what the label does.
