## Core purpose

SDeltaPhi-63 defines the **Theory-to-Module Conversion Protocol**.

This package converts external theories, philosophies, religions, ethical systems, political ideologies, governance frameworks, AI alignment theories, and user-created frameworks into AI-readable conditional detection modules.

The goal is not to adopt or reject a theory as a total worldview. The goal is to extract what the theory detects, when it should be activated, what it outputs, what it hides, where it overbinds, what remains UMR, and which downstream SDeltaPhi modules should be called.

## Skill inclusion

This package includes **SKILL.md** as a pilot Agent Skill-style file.

The `SKILL.md` file is designed to let an AI agent recognize when theory-to-module conversion should be invoked and apply the conversion procedure directly.

## Core declaration

A theory should not be treated first as a worldview to adopt or reject.

A theory can be converted into a conditional detection module.

A converted theory is not final truth. It is an operational detector with scope, triggers, outputs, limits, UMR, and failure modes.

- final truth detector,
- ideology adoption engine,
- theory debunking machine,
- total worldview generator,
- proof that one theory explains everything,
- replacement for domain expertise,
- tool for intellectual colonization.
