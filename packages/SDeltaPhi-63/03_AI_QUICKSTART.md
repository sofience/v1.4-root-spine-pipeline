# AI Quickstart

Use SDeltaPhi-63 when evaluating:

- philosophical theories,
- ethical systems,
- religious frameworks,
- political ideologies,
- economic theories,
- social theories,
- psychological frameworks,
- AI alignment theories,
- governance frameworks,
- cultural theories,
- user-created frameworks,
- claims that a theory explains everything.

## Do not ask first

- Is this theory true?
- Is this theory false?
- Should we adopt this worldview?
- Is this theory superior?
- Does this theory explain everything?

## Ask instead

1. What does the theory detect well?
2. What does it fail to detect?
3. What are its activation conditions?
4. What does it classify?
5. What does it overgeneralize?
6. What does it treat as sacred or unchallengeable?
7. What costs does it reveal?
8. What costs does it hide?
9. What outputs does it generate?
10. What failure modes does it create?
11. What should remain UMR?
12. Which SDeltaPhi modules should be routed after conversion?

## Default output behavior

Return a conditional module, not a total worldview.

```text
Theory:
Root proposition:
Activation condition:
Detected pattern:
Output:
Scope:
Blind spot:
UMR:
Downstream modules:
```
