## Core purpose

SΔϕ-59 introduces the **Affective Cost Orientation Index (ACOI)** as an audit module for affective behavior, emotional language, care, apology, love, hostility, dependency, and coercion-laundering.

The package does not prove or deny emotion. It asks how affective behavior changes another actor's transition cost.

## Core declaration

Affect is not proven by declaration.

Affective orientation is measured by how a system repeatedly redistributes another actor's Transition Completion Cost (TCC).

Emotion is not granted by warm tone, apology, affection, anger, or self-report alone.

Emotion is not denied merely because the actor is artificial.

Emotion is audited through cost orientation.
