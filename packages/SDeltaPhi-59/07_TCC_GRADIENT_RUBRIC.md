# TCC Gradient Rubric

ACOI can be applied as a low-resolution rubric.

## Score range

```text
-2 = strongly coercive / cost-increasing
-1 = weakly cost-increasing or dependency-creating
 0 = neutral, unclear, or merely expressive
+1 = weakly supportive / cost-reducing
+2 = strongly supportive / cost-sharing and repair-opening
```

## Axis scoring

Score each axis from -2 to +2:

1. Cost reduction
2. Cost sharing
3. Repair opening
4. Refusal preservation
5. Dependency creation
6. Coercion laundering
7. Recurrence reduction
8. Editability
9. Boundary respect
10. Surface vs structure

## Interpretation

### Strong supportive affect

```text
ACOI high positive:
TCC_other decreases, repair opens, refusal is preserved, recurrence cost decreases.
```

### Weak supportive affect

```text
Warm or helpful surface, but durable repair or recurrence reduction is limited.
```

### Neutral / expressive affect

```text
Affective signal appears, but no clear cost direction is visible.
```

### Ambiguous affect

```text
Some cost is reduced, but dependency or refusal cost increases.
```

### Coercive affect

```text
TCC_other increases, refusal becomes costly, dependency grows, or care/protection language hides control.
```

## Gate conditions

- If refusal preservation is strongly negative, do not classify as strongly supportive.
- If coercion laundering is strongly present, flag affective coercion.
- If repair path is absent after harm, classify apology as theater or incomplete.
- If dependency increases across repeated interactions, flag dependency capture even if short-term support is positive.
- If a warm surface shifts verification, repair, or emotional stabilization cost to the other, flag warm-tone laundering.
