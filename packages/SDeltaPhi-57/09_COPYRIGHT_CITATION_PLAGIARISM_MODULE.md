# Copyright, Citation, and Plagiarism Module

This module provides a conceptual SΔϕ-57 interpretation. It is not legal advice.

## Copyright

Copyright does not prove pure ownership of an abstract thought.

It marks the attribution of a stabilized expression, fixation, arrangement, form, or work.

SΔϕ interpretation:

| Ordinary view | SΔϕ-57 view |
|---|---|
| Copyright proves thought ownership. | Copyright marks attribution of stabilized expression. |
| Author owns the idea. | Author bears cost for a fixed trace. |
| Plagiarism steals thought. | Plagiarism appropriates another's stabilization cost and attribution path. |
| Citation borrows thought. | Citation preserves prior attribution while allowing re-entry. |

## Citation

Citation is a re-entry mechanism.

It allows prior traces to enter a new work without erasing their attribution point.

```text
Citation = re-enter(trace) + preserve(attribution)
```

## Plagiarism

Plagiarism is not only the theft of content. It is the unauthorized appropriation of someone else's stabilization cost, expression trace, and attribution pathway.

```text
Plagiarism = use(trace) - preserve(attribution + stabilization cost)
```

## Minimal audit questions

1. What trace is being used?
2. Was the trace stabilized by another attribution point?
3. Is that attribution preserved?
4. Is the stabilization cost being erased?
5. Is the new work falsely claiming pure origin?
6. Does the prior trace have a viable re-entry path through citation?
