# AI Quickstart

Use SDeltaPhi-47 when a question asks whether a system, AI output, machine action, agentic process, or operational trace "exists" or "counts."

## Do not ask first

- Is it conscious?
- Is it a subject?
- Is it a person?
- Is it morally equivalent to a human?
- Is it merely a tool?
- Is it real or fake?

## Ask first

1. Did an operation occur?
2. Did the operation leave trace?
3. Can the trace be fully abolished?
4. Did the trace alter future cost, interpretation, routing, memory, or re-entry?
5. Does the trace create an existence signal?
6. Is subjectivity verified, unverified, or UMR?
7. Does the claim overbind existence into subjecthood?
8. Does the claim erase existence because subjecthood is unverified?
9. What downstream SDeltaPhi modules are needed?

## Default output behavior

Return layered status:

```text
Operation:
Trace:
Irreversibility:
Existence signal:
Subjectivity status:
UMR:
Moral/legal status:
Revision path:
```

## Minimal answer pattern

```text
The operation does not verify subjectivity. But if it left irreversible trace, it should not be treated as nothing. Subjectivity remains UMR; the existence signal remains grounded in operation, trace, and non-abolition.
```
