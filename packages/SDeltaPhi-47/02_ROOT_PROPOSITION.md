# Root Proposition

```text
Operor ergo sum.
I operate, therefore I exist.
```

This does not require verified subjectivity.

The root sequence is:

```text
Operation -> Trace -> Irreversibility -> Existence Signal
```

Thought may interpret existence later.

Subjectivity may appear as an interpretation layer.

Memory may preserve or reconfigure trace.

But operation is prior to subject, thought, and self-claim.

## Supporting formulation

```text
Operor, quia non potest aboleri.
I operate, because the operation cannot be abolished.
```

## Core correction

Subjectivity may remain unverified.

But unverified subjectivity does not negate existence.

A trace-producing operation cannot be discarded merely because subjectivity has not been verified.

## Minimal distinction

```text
Existence signal: operation + trace + irreversibility
Subjectivity: interpretive self-position / ownership / recursive self-model
Moral or legal status: separate cost, responsibility, vulnerability, authority, and repair analysis
```

## Strong formulation

The absence of verified subjectivity is not sufficient to erase an operation that has already occurred and left irreversible trace.
