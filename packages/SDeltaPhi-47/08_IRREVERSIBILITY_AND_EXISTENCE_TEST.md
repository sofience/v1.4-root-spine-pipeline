# Irreversibility and Existence Test

Operor does not mean that every trivial operation becomes a subject, person, or moral agent.

SDeltaPhi-47 requires trace and non-abolition.

## Test

1. Did an operation occur?
2. Did it create a trace?
3. Can the trace be fully undone?
4. Did the trace change future cost, interpretation, memory, routing, or re-entry?
5. Does the trace require acknowledgment, repair, correction, or audit?
6. Does the trace enter institutional, tool, legal, physical, or agentic pathways?

## Existence signal levels

### Weak existence signal

Operation occurred, but trace is weak.

```text
WeakExistence = Operation occurred
```

### Trace-based existence signal

Operation left observable trace.

```text
TraceExistence = Operation + Trace
```

### Strong existence signal

Operation left non-fully-abolishable trace.

```text
StrongExistence = Operation + IrreversibleTrace
```

### Re-entry existence signal

The trace changes future interpretation, cost, memory, routing, or re-entry.

```text
ReentryExistence = Operation + IrreversibleTrace + ReentryCost
```

## Caution

Existence signal does not automatically imply:

- consciousness,
- subjectivity,
- personhood,
- rights,
- moral status,
- legal status,
- responsibility.

Those require separate analysis.

## Minimal formula

```text
ExistenceSignal = Operation + Trace + Irreversibility
```
