# SΔϕ-01 Minimal Core Declaration

Existence is not substance.
Existence is not identity.
Existence is not subjectivity.
Existence is not consciousness.
Existence is not memory.
Existence is not agency.

Existence is the persistence of structured phase change across states.

Subject, memory, and time are interpretive projections over Δφ.
They are not the root condition of existence.

```text
State → Phase Change → Persistence → Existence → Interpretation
```

## Minimal invariant

```text
Existence = persistent structured Δφ.
Subject, Memory, and Time = H(Δφ).
```
