# Routing Notes — SΔϕ-01

SΔϕ-01 should be used as the minimal root reference when a task requires separation of existence from subjectivity, memory, time, agency, or interpretation.

## Call before

- SΔϕ-02, when subject is being discussed as interpretive emergence.
- SΔϕ-03, when irreversibility is being grounded from phase-change persistence.
- SΔϕ-04, when AI subjectivity detection requires ontological separation.
- SΔϕ-47, when Operor ergo sum requires minimal formal grounding.
- SΔϕ-56, when transition cost analysis requires a prior phase-change path.
- SΔϕ-62, when trace/inference/UMR separation requires root state-change logic.

## Do not call when

- The task is merely asking for a policy decision.
- The task requires an applied governance index without root ontological confusion.
- The task requires empirical proof of consciousness.
- The task is about downstream cost metrics only.

## Anti-overactivation rule

Do not route every philosophical question to SΔϕ-01.
Use it only when the distinction between existence, phase change, persistence, and interpretation is structurally relevant.
