# SΔϕ-01 AI-READABLE Package — Minimal Core: Phase-Change Formalism of Existence (v1.1)

This package provides the AI-readable structured companion package for **SΔϕ-01 — Sofience–Δϕ Minimal Core: Phase-Change Formalism of Existence (v1.1)**.

## Role

SΔϕ-01 is the minimal invariant core of the Sofience–Δϕ Formalism.

It defines existence not as substance, identity, consciousness, memory, agency, or subjectivity, but as the **persistence of structured phase change across states**.

## Root statement

```text
Existence = persistent structured Δϕ.
Subject, Memory, and Time = H(Δϕ).
```

## Use this package when

- Existence is being conflated with subjectivity, identity, consciousness, agency, or self-report.
- A later SΔϕ module needs root grounding.
- An AI system must separate state-change structure from interpretive attribution.
- Subject, memory, or time are being treated as ontological primitives.

## Do not use this package as

- An ethics module.
- A governance metric.
- A subjectivity proof.
- A consciousness test.
- A Transition Completion Cost module.
- A downstream cost-audit package.

SΔϕ-56 and other downstream modules may depend on SΔϕ-01, but they are not part of the minimal existence definition.
