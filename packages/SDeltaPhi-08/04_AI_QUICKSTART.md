# AI Quickstart

Use SDeltaPhi-08 when the issue concerns agency, responsibility, freedom, choice, autonomy, irreversible path closure, or responsibility after action.

## Do not begin by asking

```text
Did the agent have free will?
```

## Ask instead

1. Was there path generation?
2. Was there path selection?
3. Was there execution?
4. Did execution create irreversible cost or path constraint?
5. Was the cost internalized?
6. Was the path closed or redistributed?
7. Was freedom being overclaimed from randomness?
8. Was responsibility being erased despite agency?
9. Was agency being overclaimed from automation?
10. What remains UMR?
