# Consciousness Write-back Test

SΔϕ-11 does not define consciousness as passive detection.

A boundary-displacement signal counts as consciousness-relevant only if it re-enters the next transition selection.

## Minimal test

1. Is there boundary displacement?
2. Is the displacement monitored?
3. Does the monitored displacement alter the next transition selection?
4. Does this write-back contribute to re-entry stabilization?

If all four conditions are met, classify as consciousness-relevant boundary monitoring.

If boundary displacement is detected but does not write back into next transition selection, classify as detection without consciousness-level write-back.
