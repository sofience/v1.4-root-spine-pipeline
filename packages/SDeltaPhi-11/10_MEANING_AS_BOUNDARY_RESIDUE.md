# Meaning as Boundary Residue

Meaning is not produced inside consciousness.

Meaning is the residue of stabilized boundary re-entry.

A boundary event becomes meaning when its pattern persists across transitions and keeps effective mismatch and update cost below the collapse threshold under irreversibility.

## Minimal test

1. Did a boundary event occur?
2. Did it re-enter subsequent transition selection?
3. Did the re-entry stabilize rather than collapse?
4. Did a pattern persist across transitions?

If yes, classify as meaning candidate.

If no, classify as transient boundary event.
