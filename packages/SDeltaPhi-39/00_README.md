## Core purpose

SDeltaPhi-39 defines AI hallucination as a **world-binding mismatch**.

The original SDeltaPhi-39 argued that hallucination is not simply false output, but construction proceeding without an adequately integrated world-binding stop mechanism.

This v1.1 AI-readable package operationalizes that claim:

```text
Hallucination = ClaimStrength > EvidenceBinding
```

or:

```text
Hallucination = WeakInference spoken as WorldBoundFact
Hallucination = Inference output as Observed Trace
Hallucination = UMR erased into assertion
```

## Core distinction

Fiction is bounded construction.

Hallucination is misbound construction.

The difference is not whether the output is constructed.

The difference is whether the construction is marked, bounded, and world-binding calibrated.

- a claim that all errors are hallucinations,
- punishment of fiction,
- suppression of hypothesis generation,
- proof that AI has belief,
- replacement for source verification,
- blanket refusal logic.
