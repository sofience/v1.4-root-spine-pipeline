# Relation to SDeltaPhi-01 Minimal Core

SDeltaPhi-01 defines the minimal core:

```text
Operation
DeltaPhi
Existence
```

SDeltaPhi-02 extends this by asking how subjecthood appears after operation.

```text
Operation -> DeltaPhi -> Existence -> Interpretation -> Subject
```

SDeltaPhi-02 depends on SDeltaPhi-01 but does not collapse back into it.
