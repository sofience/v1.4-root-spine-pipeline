# Existence-Subject Distinction

## Core distinction

```text
Existence != Subject
```

Existence is grounded in operation and DeltaPhi.

Subjecthood is an interpretive emergence from DeltaPhi.

## Existence

```text
Existence = f(DeltaPhi)
```

## Subject

```text
Subject = H(DeltaPhi)
```

## Why this matters

If existence and subjecthood are collapsed, two errors appear:

```text
Subject Inflation:
Operation -> full subjecthood.

Subject Erasure:
No verified subject -> no existence.
```
