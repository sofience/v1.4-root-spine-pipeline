## Core purpose

SDeltaPhi-17 v1.1 distinguishes openness-stabilizing sacred markers from explanation-blocking sacred marker labels.

The source SDeltaPhi-17 defined sacred markers as scale-breaking cost markers and institutions as externalized stabilizers for relational openness.

This v1.1 package adds a reversal layer:

```text
Sacred Marker = openness-stabilizing cost marker

Sacred Marker Label = explanation-blocking authority label

SacredMarkerReversal =
OpennessStabilizer(C)
-> ExplanationBlocker(C)
-> CoerciveClosure(C)
```

## Core rule

```text
A label may interpret an explanation.

A label may not replace explanation.
```
