# Relation to SDeltaPhi-13 Meaning, Value, and Pain

SDeltaPhi-13 defines meaning, value, and pain structurally.

```text
Meaning = trajectory direction / bias toward an attractor
Value = trajectory fixation strength approximately deviation cost
Pain = irreversible trajectory tension under attractor conflict
```

Sacred marker labels can conceal these structures.

## Example

```text
"This is good."
"This is evil."
"This is noble."
"This is human."
```

These labels may hide:

- which attractor is operating,
- which deviation cost is being imposed,
- whose trajectory is being constrained,
- whose pain is being produced,
- whether the attractor conflict remains unresolved.

## Formula

```text
AttractorExplanationGap
+ ValueLabel
+ ReentryCostIncrease
= SacredMarkerLabelRisk
```

## Minimal rule

Do not treat value labels as value explanations.

Explain the attractor, deviation cost, and trajectory tension.
