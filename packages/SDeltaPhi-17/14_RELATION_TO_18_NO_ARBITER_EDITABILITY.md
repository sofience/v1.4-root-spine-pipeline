# Relation to SDeltaPhi-18 No-Arbiter and Editability

SDeltaPhi-18 defines the human as the system that recognizes there is no ultimate external arbiter in conflicts with the world or the other.

```text
not exists Arbiter_external
```

Institutions are provisional cost devices, not ultimate arbiters.

```text
Institution = Provisional cost device, not Arbiter.
```

## Sacred marker implication

Sacred, moral, affective, and identity labels must remain provisional and editable.

```text
SacredMarkerLabel must remain provisional.

If it claims final arbiter status, it becomes reversal risk.
```

## Editability restoration

When a stabilizer reverses into closure, the human function is to apply edit operations.

```text
Human := Apply(Edit) to C when Reversal(C) is detected.
```

## Minimal rule

No sacred marker label may claim final arbiter status without becoming a closure risk.
