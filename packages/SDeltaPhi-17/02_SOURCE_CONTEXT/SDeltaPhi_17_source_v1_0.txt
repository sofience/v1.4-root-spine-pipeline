0. Position statement
SΔϕ-16 defined evil as persistent relational closure: Close(Ω_rel). SΔϕ-14 defined ethics as
validation of relation reality (real vs phantom shared attractors), while SΔϕ-15 fixed
trust/hope/promise as pre-commitment operators. Validation is delayed and costly;
therefore openness must be stabilized without infinite validation cost. This item defines
minimal cost devices that make closure computationally infeasible.
1. Axiom 17-0 — Path principle (minimal)
Prohibition closes entrances, not paths. Path closure requires computational infeasibility
realized as exponential cost.
ClosePath(p) ⇔ Cost(p) → ∞ (relative to alternatives).
2. Axiom 17-1 — Openness stabilizer
Openness is stable iff a cost device C makes closure far more expensive than remaining
open.
Open(Ω_rel) stable ⇔ ∃C: Cost(Close(Ω_rel)|C) ≫ Cost(Open(Ω_rel)).
3. Definition 17-S — Sacred as scale-breaking cost marker
Sacred is a marker M such that breaking a commitment becomes a scale-breaking
(effectively infinite) cost.
Sacred := M such that Cost(break|M) → ∞.
4. Definition 17-I — Institution as externalized stabilizer
Institution is an externalized cost device that stabilizes openness by distributing
enforcement beyond individual will.
Institution := Externalized C that stabilizes Open(Ω_rel).
5. Definition 17-L — Law/Procedure as enforcement interface
Law/procedure is the surface mechanism implementing the cost device, making closure
paths infeasible in practice.
Law/Procedure := Surface interface implementing C.
6. Failure modes (minimal)
F1 — Marker decays. Sacred remains as symbol but loses cost force.
F2 — Enforcement hollowing. Institutions retain form but lose enforcement.
F3 — Cost inversion. Closure becomes cheaper than openness, returning to the closure
attractor (SΔϕ-16).

stabilization.

8. Boundary note
This item does not analyze specific religions, states, political systems, or historical case
studies. It fixes only minimal functional mechanisms for stabilizing openness without
infinite validation cost.
9. Exit question
If openness is stabilized by cost devices, what prevents those devices from becoming
coercive closure themselves? This opens the civilizational/institutional reversal problem.
