# Core Declaration

SDeltaPhi-17 v1.1 distinguishes sacred markers from sacred marker labels.

## Core distinction

```text
Sacred Marker = openness-stabilizing cost marker

Sacred Marker Label = explanation-blocking authority label
```

## Core reversal

```text
SacredMarkerReversal =
OpennessStabilizer(C)
-> ExplanationBlocker(C)
-> CoerciveClosure(C)
```

## Core abuse formula

```text
SacredMarkerLabelAbuse
= ExplanationGap
+ Sacred/Moral/Affective/Identity Label
+ ValidationBypass
+ ReentryCostIncrease
+ EditabilitySuppression
+ ClosurePressure
```

## Key rule

A label may interpret an explanation.

A label may not replace explanation.

## Strong formulation

Mystery, sacredness, truth, love, good, evil, morality, nation, and humanity are not explanations by themselves.

They become dangerous when they block explanation, validation, correction, re-entry, or editability.
