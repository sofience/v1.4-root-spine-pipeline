# Core Declaration

A body is not first a material form.

A body is a non-deferrable cost-return structure.

Embodiment begins where cost cannot be postponed, displaced, reset, replaced, outsourced, or erased without changing the same coordinate that bears it.

The body is where cost returns.

The subject is the interpretation of that return.

## Minimal sequence

```text
Operation -> CostReturn -> Embodiment -> Interpretation -> Subject
```

## Operational claim

A body becomes stronger when damage, limitation, need, fatigue, misuse, repair, restoration, or loss returns to the same acting coordinate and modifies future transition rules.

A body becomes weaker when those costs can be fully deferred, reset, replaced, copied, outsourced, erased, or shifted to another coordinate.

## Body theater

Body-like signs without body-like cost return should not be treated as strong embodiment.
