# Disclosure-to-Re-entry TCC

Disclosure is not re-entry.

SDeltaPhi-56 v1.3 measures the cost from disclosure to re-entry.

## Formula use

```text
Disclosure -> repair path -> re-entry -> recurrence-cost reduction
```

## Audit questions

2. Did it open a repair path?
3. What friction coefficients block re-entry?
