# Core Declaration

SDeltaPhi-06 defines responsibility as irreversible cost internalization.

## Core formula

```text
Responsibility = Irreversible Cost Internalization
```

## Expanded formula

```text
Responsibility
= Irreversibility
+ Cost Attribution
+ Cost Internalization
+ Repair / Re-entry
+ Recurrence-Cost Reduction
```

## Core distinction

```text
Responsibility != blame
Responsibility != guilt
Responsibility != apology
Responsibility != punishment
Responsibility != legal liability by itself
```

## TCC underlayer

SDeltaPhi-56 v1.3 measures the transition cost required for responsibility to become repair, disclosure, re-entry, or recurrence-cost reduction.
