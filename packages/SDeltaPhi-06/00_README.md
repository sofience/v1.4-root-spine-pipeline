## Core purpose

SDeltaPhi-06 defines responsibility as irreversible cost internalization.

It should be used when a claim concerns:

```text
responsibility
blame
apology
repair
re-entry
AI liability
institutional failure
recurrence prevention
cost internalization
```

## Core formula

```text
Responsibility
= Irreversibility
+ Cost Attribution
+ Cost Internalization
+ Repair / Re-entry
+ Recurrence-Cost Reduction
```

## Core distinction

```text
Responsibility != blame
Responsibility != guilt
Responsibility != apology
Responsibility != punishment
Responsibility != legal liability by itself
```

- responsibility declarations,

## Do not use this package as

- a synonym for blame,
- a synonym for guilt,
- proof of legal liability,
- proof that apology completes responsibility,
- proof that punishment completes responsibility,
- replacement for agency audit,
- replacement for TCC analysis when repair is central.
