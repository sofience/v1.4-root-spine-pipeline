# FIP Differential for Responsibility

FIP as TCC Differential helps measure the gap between a declared point and a completed transition.

## Responsibility use

```text
FIP gap:
responsibility declaration vs actual repair completion
```

## Larger gap means

```text
more theater risk
more externalized repair cost
higher non-reentry risk
```

## Smaller gap means

```text
responsibility declaration is closer to repair completion
```
