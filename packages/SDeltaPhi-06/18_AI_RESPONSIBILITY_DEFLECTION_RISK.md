# AI Responsibility Deflection Risk

AI responsibility deflection occurs when responsibility is pushed onto AI to shield operators, platforms, institutions, or tool-chain designers.

## Example

```text
The AI made the decision.
The model hallucinated.
The system did it autonomously.
```

## Correction

Audit the full path:

```text
model
operator
platform
tool chain
deployment default
policy
user action
governance layer
```
