# Irreversible Cost Produced

Responsibility begins when an action, omission, default, output, decision, or system path produces cost that cannot be fully undone.

## Examples

```text
harm to a person
misinformation spreading
loss of trust
reputation damage
resource depletion
failed AI output causing downstream work
default path causing hidden switching cost
```

## Audit questions

1. What cost was produced?
2. Is it reversible or irreversible?
3. Who currently bears it?
4. What future constraints did it create?
