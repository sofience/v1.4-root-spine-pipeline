# Hope as Closure Suspension

Hope is suspension of closure.

The system refrains from declaring the future feasible path set closed.

```text
Hope := Suspend_Closure(Ω_future)
```

Despair is closure of the future feasible path set.

```text
Despair := Close(Ω_future)
```

## Function

Hope does not guarantee future success.

Hope keeps future transitionability open under uncertainty.

## Boundary

Hope is not optimism.

Hope is not belief that success will happen.

Hope is refusal to prematurely close Ω_future.
