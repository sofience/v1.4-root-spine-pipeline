# Failure Mode Map

| Operator | Failure mode | Structural collapse |
|---|---|---|
| Trust | Betrayal | Collapse of relational persistence |
| Belief | Illusion | Persistent mismatch with world Δϕ |
| Hope | Despair | Closure of Ω_future |
| Promise | Break(Promise) | Collapse(A_shared) |

## Diagnostic rule

Do not classify all failures as betrayal.

- Trust failure → betrayal.
- Belief failure → illusion.
- Hope failure → despair.
- Promise failure → shared-attractor collapse.
