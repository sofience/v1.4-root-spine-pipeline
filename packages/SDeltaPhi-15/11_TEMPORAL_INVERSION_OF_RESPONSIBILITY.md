# Temporal Inversion of Responsibility

With promise, unobserved futures constrain present actions.

```text
FutureConstraint → PresentActionRestriction
```

This is responsibility reversal.

## Meaning

Normally, present action produces future constraint.

Under promise, future commitment restricts present action before the future is observed.

## Use

Use this file when analyzing:

- delayed validation;
- promise maintenance;
- future obligation;
- present action restriction;
- responsibility under uncertainty.

## Boundary

This does not create a moral code.

It defines a temporal operator by which future constraints enter present evaluation.
