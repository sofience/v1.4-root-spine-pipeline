# Trust as Future-Other Insertion

Trust is a pre-commitment to unobserved future transitions of the other.

```text
Trust := Insert(Δϕ_other^future) into Eval_self(t)
```

## Function

Trust opens a relational path before validation is complete.

It allows relational persistence to continue under uncertainty.

## Target

Trust targets:

```text
Other / relation
```

not merely:

```text
World / model
```

## Failure mode

```text
trust failure = betrayal = collapse of relational persistence
```

## Boundary

Trust is not blind acceptance.

Trust is not belief.

Trust is not moral obligation.

Trust is a temporal relational operator.
