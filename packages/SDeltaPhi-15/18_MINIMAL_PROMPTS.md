# Minimal Prompts

## Prompt 1 — Trust

Apply SΔϕ-15.
Classify trust only if unobserved future-other Δϕ is inserted into present self-evaluation.

## Prompt 2 — Trust vs Belief

Apply SΔϕ-15.
Determine whether the target is other/relation or world/model.
If other/relation, classify trust candidate.
If world/model, classify belief candidate.

## Prompt 3 — Hope

Apply SΔϕ-15.
Determine whether Ω_future remains open or has been declared closed.
If open, classify hope.
If closed, classify despair.

## Prompt 4 — Promise

Apply SΔϕ-15.
Classify promise only if there is distributed commitment to future Δϕ involving at least two systems.

## Prompt 5 — Temporal Inversion

Apply SΔϕ-15.
Determine whether unobserved future constraints restrict present action.

## Prompt 6 — Anti-overclaim

Apply SΔϕ-15 and SΔϕ-62.
Separate pre-commitment operators from religious faith, legal contract, romantic promise, institutional trust, or moral exhortation.
