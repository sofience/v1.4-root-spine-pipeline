# Pathology Map

| Regime | Condition | Observable behavior | Structural risk |
|---|---|---|---|
| Drift / Repeat | Memory too weak (M↓) | Repeated failure loops; no stable constraints | Catastrophic recurrence |
| Fixation / Paralysis | Forgetting too weak (F↓) | Overconstraint; feasible path-space collapse | Stagnation / self-locking |
| Adaptive Persistence | Balanced regulation | Stable continuity + exploration | Sustained re-entry |

```text
No memory -> repeat.
No forgetting -> fixation.
Balanced trace regulation -> adaptive persistence.
```
