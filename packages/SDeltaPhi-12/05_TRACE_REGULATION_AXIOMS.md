# Trace Regulation Axioms

## Axiom 1 — Trace Necessity

If irreversible traces do not re-enter selection, catastrophic repeats occur.

## Axiom 2 — Trace Overload

If all traces re-enter, feasible path space collapses. This regime appears as fixation or paralysis.

## Axiom 3 — Regulation

Persistence requires regulated effective memory:

```text
M_eff_t := C(H_M(Δϕ_0:t))
```

## Axiom 4 — Freedom Coupling

Forgetting is the minimal condition for freedom under irreversibility. Excessive forgetting increases repeat risk.

## Trace Overload Formula

```text
|M_t| ↑↑ ⇒ |Ω_{t+1}| ↓ → 0
```
