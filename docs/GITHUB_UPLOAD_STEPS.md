# GitHub Upload Steps

1. Create a new GitHub repository.
2. Unzip this seed repository package.
3. Upload the root files and folders:
   - README.md
   - LICENSE.md
   - llms.txt
   - SERIES_INDEX.json
   - SERIES_DOI_MAP.json
   - SDELTA_ACCESS_PROTOCOL.yaml
   - SDELTA_MODULE_ROUTING.yaml
   - packages/
   - indexes/
   - routing/
   - docs/
4. Prefer uploading extracted package folders, not just archives.
5. Put original ZIP archives in releases or keep them under /archives/ if desired.
6. In each Zenodo record, optionally add the GitHub repository as:
   - Related identifier: Is supplemented by
7. In GitHub README, keep Zenodo as canonical citation layer.

## Suggested repository description

AI-readable mirror and routing layer for selected SΔϕ (Sofience-DeltaPhi) Formalism packages. Zenodo remains the canonical DOI/citation archive.
